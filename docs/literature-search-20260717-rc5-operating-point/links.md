## Citation cautions

- Papers 10–12 are mandatory when discussing unlabeled/transductive threshold inference.
- Papers 13–15 are mandatory when using Neyman–Pearson, risk-control, guarantee, or optimality terminology.
- Papers 4–6 and 16 should inform the current strong-detector baseline discussion.
- Paper 3 should inform evaluation-protocol positioning.
- Preprints (1, 11, and 24) must be labeled as preprints unless their venue status is separately verified from an allowed source before submission.
